Concepts Of WWW

  • It Stands For World Wide Web
  • It is Basically a system of internet servers that specially formatted Documents
  • It is collection of text pages ,videos,digital photographs and animation we can access over the internet
  • User access the world wide web facilities via client called web browser

HTTP Protocol

  1. HTTP stands for Hyper Transfer Protocol
  2. HTTP is Based on the client-server architecture model and a stateless request/response protocol
  3. 404 File Not Found is a common HTTP status code
